# Screenshots Directory

Place your application screenshots here for the homepage hero slider.

## Required Files

1. **desktop-view.png** - Desktop/laptop view of your application
   - Recommended size: 1920x1200px or 1600x1000px
   - Format: PNG or JPG
   - Shows the full app interface on desktop

2. **mobile-view.png** - Mobile/PWA view of your application
   - Recommended size: 750x1334px (iPhone) or similar mobile aspect ratio
   - Format: PNG or JPG
   - Shows the mobile/PWA app interface

## Tips for Best Results

- Use high-quality screenshots with clear text
- Capture actual app content (not placeholder screens)
- Ensure images are optimized for web (compressed but still sharp)
- Aspect ratio for frame: 16:10 (width:height)
- The slider will auto-scale images to fit the frame
